# Duck Treasure Hunt
A fast-paced arcade-style game built for the Game Boy Advance using the Butano engine.  
Collect treasures, use speed boosts, and avoid enemies as the difficulty increases!

## How to Play
- Move with the D-Pad
- Press A for a speed boost (3 per game)
- Catch the ladybug to increase your score
- Avoid the car or lose points
- Watch out for the fox after level 15!
