# Chased

This game is a ground breaking game that is fun to play, where you have to collect tresures while being chased by an enemy.

Over time the enemy will get faster and faster making it harder to play.
There is an option to boost but use it wisely becouse you only hace 3 per game.

Instructions/Controls:
- Move: D-Pad
- Boost: A
- Reset Game: Start

Have fun and don't let the ops catch you.

Play here: https://sodatax.github.io/collectathon/
