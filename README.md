# Collectathon (GameBoy Advance)

A simple game built using **Butano**, a C++ high-level GBA engine.
Collect coins, avoid moving walls, to boost for high score!

##  About

This is a GameBoy Advance game written in C++ using the **Butano** engine.  
The player moves a square with the D-pad, collects coins, and avoids obstacles.

# Features
-  Coin collection with scoring
-  Moving wall obstacles
-  Coin sound effect
-  Screen wrap-around movement
-  Speed boost ability (3 is the limit!)
-  High score tracking
-  GBA controls (D-pad + A + Start)
